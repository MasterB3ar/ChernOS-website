Get Started

Build and boot in minutes

Build the ISO

Requires Nix with flakes enabled.

nix build .#iso -L --print-build-logs

The resulting ISO is produced by your flake output. Write it to USB (or boot in a VM).

Recommended VM Settings

For VirtualBox / similar:

  • VRAM128MB+ if possible
  • 3D AccelOn (if stable)
  • Resolution1920×1080 for best layout

Controls

Common defaults (your build may differ if you changed hotkeys):

Install (Calamares): Ctrl+Alt+I / Super+I Electron suite: Ctrl+Alt+E / Super+E Night Shift: Ctrl+Alt+Shift+N
